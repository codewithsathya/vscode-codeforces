## ğŸš€ Installation

1. Open **VS Code**.
2. Go to **Extensions** (`Ctrl + Shift + X`).
3. Search for **Codeforces VS Code Extension**.
4. Click **Install**.
5. Start solving problems directly in VS Code!

## ğŸ› ï¸ Usage

- **Browse problems** from the sidebar.
- **Select categories/tags** to filter problems.
